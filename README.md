# shell-script-in-out-different-file-formats
This project is for personal learning purpose. The project will be an experimental studying project for shell reading & searching & outputting. The input content is randomly generated by AI in different formats (same contents. Just file type is different.)
## Each file includes combinations of(AI Generated):
-  names and emails
-  cities\n
-  numbers
-  Unicode text
-  sentences
-  code snippets
-  URLs
-  comma-separated tags
### File formats are in: CSV, JSON, .xlsx, Markdown dataset, Plain text dataset.
### In the first version I will focus on making read/search/output work, and then I might add more advanced functions and try automation stuffs.
